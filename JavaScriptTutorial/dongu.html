Farklı Döngü Çeşitleri

JavaScript farklı döngü türlerini destekler:

  • for - bir kod bloğundan birkaç kez geçer

  • for / in - nesnenin özellikleri arasında dolaşır

  • for / of - yinelenebilir bir nesnenin değerleri arasında geçiş yapar

  • while - belirtilen bir koşul geçerliyken bir kod bloğundan geçer

  • do / while - ayrıca belirtilen bir koşul geçerliyken bir kod bloğunda da dolaşır

    • For döngüsü

    For döngüsü aşağıdaki sözdizimine sahiptir:

    for (ifade 1; ifade 2; ifade 3) {

    // yürütülecek kod bloğu

    }

  • İfade 1, kod bloğunun yürütülmesinden önce (bir kez) yürütülür.

  • İfade 2, kod bloğunu yürütme koşulunu tanımlar.

  • İfade 3, kod bloğu yürütüldükten sonra (her seferinde) yürütülür.

    • for (i = 0; i <5; i ++) {

    text + = "Sayı" + i + "
    ";

    }

    Yukarıdaki örnekte şunları okuyabilirsiniz:

  • İfade 1 döngü başlamadan önce bir değişken ayarlar (var i = 0).

  • İfade 2, döngünün çalışması için koşulu tanımlar (i 5'ten küçük olmalıdır).

  • 3. ifade, döngüdeki kod bloğu her yürütüldüğünde bir değeri (i ++) artırır.

    • İfade 1

    Normalde döngüde kullanılan değişkeni başlatmak için 1. ifadeyi kullanırsınız (i = 0).

    Bu her zaman böyle değildir, JavaScript umursamaz. Bildirim 1 isteğe bağlıdır.

    JavaScript While Döngüsü

    Döngüler, belirtilen bir koşul geçerli olduğu sürece bir kod bloğu yürütebilir.

    While Döngüsü

    While döngüsü, belirtilen bir koşul geçerli olduğu sürece bir kod bloğundan geçer.

    Sözdizimi

    while (koşul) {

    // yürütülecek kod bloğu

    }

    Aşağıdaki örnekte, döngüdeki kod çalışacak, bir değişken (i) 10'dan az olduğu sürece tekrar tekrar:

    (i <10) {

    text + = "Sayı" + i;

    i ++;

    }

    Koşulda kullanılan değişkeni artırmayı unutursanız, döngü asla sona ermez. Bu, tarayıcınızı kilitler.

    Do / While Döngüsü

    Do / while döngüsü, while döngüsünün bir çeşididir.

    Bu döngü, koşulun doğru olup olmadığını kontrol etmeden önce kod bloğunu bir kez yürütür, ardından koşul doğru olduğu sürece döngüyü tekrarlar.

    Sözdizimi

    do {

    // yürütülecek kod bloğu

    }

    while (koşul);

    Aşağıdaki örnekte bir do / while döngüsü kullanılmıştır. Koşul yanlış olsa bile döngü her zaman en az bir kez yürütülür, kod bloğu koşul test edilmeden önce yürütüldüğünden:

    do {

    text + = "Sayı" + i;

    i ++;

    }

    (i <10);

    Uyarı!

    Koşulda kullanılan değişkeni artırmayı unutmayın, aksi takdirde döngü asla bitmez!
