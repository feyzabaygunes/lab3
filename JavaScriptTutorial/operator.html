JavaScript Operatörler

Değişkenlere değerler atayın ve bunları birlikte ekleyin:

var x = 5; // assign the value 5 to x

var y = 2; // assign the value 2 to y

var z = x + y; // assign the value 7 to z (x + y)

Atama operatörü ( =) bir değişkene bir değer atar.

var x = 10;

Artı operatörü ( +) değeri ekler:

var x = 5;

var y = 2;

var z = x + y;

Çarpma operatörü ( *) sayıları çarpar.

var x = 5;

var y = 2;

var z = x * y;

JavaScript Atama İşleçleri

Atama işleçleri, JavaScript değişkenlerine değerler atar.

Operatör Örnek Aynı
= x = y x = y
+ = x + = y x = x + y
- = x - = y x = x - y
* = x * = y x = x * y
/ = x / = y x = x / y
% = x% = y x = x % y
** = x ** = y x = x ** y

Toplama atama işleci (+ =) bir değişkene değer ekler.

var x = 10;

x + = 5;

JavaScript Dize İşleçleri

+ İşleci, dizeleri eklemek (birleştirmek) için de kullanılabilir.

var txt1 = "John";

var txt2 = "Doe";

var txt3 = txt1 + "" + txt2;


Txt3 sonucu şöyle olur:

John Doe

+ = Atama işleci, dizeleri eklemek (birleştirmek) için de kullanılabilir:

var txt1 = "Ne çok";

txt1 + = "iyi günler";


Txt1 sonucu şöyle olur:

Ne güzel bir gün

Dizelerde kullanıldığında + işleci, birleştirme işleci olarak adlandırılır.

Dize ve Sayı Ekleme

İki sayı eklemek toplamı döndürür, ancak sayı ve dize eklemek bir dize döndürür:

Örnek

var x = 5 + 5;

var y = "5" + 5;

var z = "Merhaba" + 5;


X, y ve z'nin sonuçları şöyle olacaktır:

10

55

Hello5

Bir sayı ve dize eklerseniz, sonuç bir dize olur!
