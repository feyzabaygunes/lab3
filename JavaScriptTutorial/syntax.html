JavaScript Sözdizimi

JavaScript sözdizimi, JavaScript programlarının nasıl oluşturulduğu ile ilgili kurallar dizisidir:

var x, y, z; // How to declare variables

x = 5; y = 6; // How to assign values

z = x + y; // How to compute values

JavaScript Değerleri

JavaScript sözdizimi iki tür değer tanımlar: Sabit değerler ve değişken değerler.

Sabit değerlere değişmez değerler denir . Değişken değerlerine değişken denir .

JavaScript Değişmezleri

Sabit değerler yazmak için en önemli kurallar şunlardır:

Sayılar ondalıklı veya ondalıksız olarak yazılır:

10.50

1001

Dizeler çift ​​veya tek tırnak içinde yazılan metindir:

"John Doe"

'John Doe'

JavaScript Değişkenleri

Bir programlama dilinde, veri değerlerini saklamak için değişkenler kullanılır .

JavaScript, değişkenleri bildirmek için varanahtar kelimeyi kullanır .

Bir eşittir işareti kullanılır atama değerlere değişkenlere.

Bu örnekte, x bir değişken olarak tanımlanmıştır. Sonra, x değeri 6'ya atanır (verilir):

var x;

x = 6;

JavaScript Operatörleri

JavaScript değerleri hesaplamak için aritmetik işleçleri ( + - * /) kullanır :

(5 + 6) * 10

JavaScript, değişkenlere değer atamak için bir atama operatörü ( =) kullanır :

var x, y;

x = 5;

y = 6;

JavaScript İfadeleri

İfade, bir değeri hesaplayan değerlerin, değişkenlerin ve işleçlerin birleşimidir.

esaplamaya değerlendirme denir.

Örneğin, 5 * 10 değeri 50 olarak değerlendirilir:

5 * 10

İfadeler ayrıca değişken değerler içerebilir:

x * 10

Değerler, sayılar ve dizeler gibi çeşitli türlerde olabilir.

Örneğin, "John" + "" + "Doe", "John Doe" olarak değerlendirilir:

"John" + " " + "Doe"

JavaScript Anahtar Kelimeleri

JavaScript anahtar sözcükleri , gerçekleştirilecek eylemleri tanımlamak için kullanılır.

var Anahtar kelime değişkenler oluşturmak için tarayıcı söyler:

var x, y;

x = 5 + 6;

y = x * 10;

JavaScript Yorumları

Tüm JavaScript ifadeleri "yürütülmez".

Çift bölü sonra Kod // veya aralarında /* ve */ bir olarak kabul edilir

Yorumlar yok sayılır ve yürütülmez:

var x = 5; // I will be executed

// var x = 6; I will NOT be executed

JavaScript Tanımlayıcıları

Tanımlayıcılar isimlerdir.

JavaScript'te tanımlayıcılar değişkenleri (ve anahtar sözcükleri, işlevleri ve etiketleri) adlandırmak için kullanılır.

Yasal adlar için kurallar çoğu programlama dilinde aynıdır.

Sonraki karakterler harf, rakam, alt çizgi veya dolar işareti olabilir.

İlk karakter olarak sayılara izin verilmez.

Bu şekilde JavaScript, tanımlayıcıları sayılardan kolayca ayırt edebilir.

JavaScript Büyük / Küçük Harfe Duyarlı

Tüm JavaScript tanımlayıcıları büyük / küçük harfe duyarlıdır .

Değişkenler lastName ve lastname, iki farklı değişkendir:

var lastname, lastName;

lastName = "Doe";

lastname = "Peterson";

JavaScript, VAR veya Var'ı var anahtar kelimesi olarak yorumlamaz .

JavaScript ve Deve Kılıfı

Tarihsel olarak, programcılar birden fazla kelimeyi bir değişken ismine birleştirmenin farklı yollarını kullanmışlardır:

Tire:

ad, soyadı, ana kart, şehirler arası.

JavaScript'te kısa çizgilere izin verilmez. Çıkarma için ayrılmıştır.

Vurgulamak:

ad_adı, soyad_adı, master_card, inter_city..

Üst Deve Kılıfı (Pascal Kılıfı):

Adı, Soyadı, MasterCard, InterCity.

Alt Deve Kılıfı:

JavaScript programcıları küçük harfle başlayan deve davası kullanma eğilimindedir:

firstName, lastName, masterCard, interCity.

JavaScript, Unicode karakter kümesini kullanır .

Unicode dünyadaki tüm karakterleri, noktalama işaretlerini ve simgeleri kapsar (neredeyse).
