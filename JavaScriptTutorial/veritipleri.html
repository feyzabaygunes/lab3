JavaScript Veri Türleri

JavaScript değişkenleri birçok veri türüne sahip olabilir: sayılar, dizeler, nesneler ve daha fazlası:

var uzunluk = 16; // Numara

var lastName = "Johnson"; // Dize

var x = {firstName: "John", soyadı: "Doe"}; // Nesne

Veri Türleri Kavramı

Programlamada veri türleri önemli bir kavramdır.

Değişkenler üzerinde çalışabilmek için tür hakkında bir şeyler bilmek önemlidir.

Veri türleri olmadan bir bilgisayar bunu güvenli bir şekilde çözemez:

var x = 16 + "Volvo";

On altıya "Volvo" eklemek mantıklı mı? Bir hata mı üretecek yoksa bir sonuç üretecek mi?

JavaScript yukarıdaki örneği şu şekilde ele alacaktır:

var x = "16" + "Volvo";

Bir sayı ve dize eklerken, JavaScript numarayı dize olarak görür.

JavaScript ifadeleri soldan sağa doğru değerlendirir. Farklı diziler farklı sonuçlar verebilir:

JavaScript:

var x = 16 + 4 + "Volvo";

Sonuç:

20Volvo

JavaScript:

var x = "Volvo" + 16 + 4;

Sonuç:

Volvo164

  • İlk örnekte, JavaScript 16 ve 4'ü "Volvo" ya ulaşana kadar sayı olarak ele alır.

  • İkinci örnekte, ilk işlenen bir dize olduğundan, tüm işlenenler dize olarak ele alınır.

    JavaScript Türleri Dinamiktir

    JavaScript'in dinamik türleri vardır. Bu, aynı değişkenin farklı veri türlerini tutmak için kullanılabileceği anlamına gelir:

    var x; // Şimdi x tanımsız

    x = 5; // Şimdi x bir sayıdır

    x = "John"; // Şimdi x bir string

    JavaScript Dizeleri

    Bir dize (veya bir metin dizesi) "John Doe" gibi bir dizi karakterdir.

    Dizeler tırnak işaretleri ile yazılır. Tek veya çift tırnak kullanabilirsiniz:

    var carName1 = "Volvo XC60"; // Çift tırnak işareti kullanma

    var carName2 = 'Volvo XC60'; // Tek tırnak kullanma

    Bir dizenin içindeki tırnak işaretlerini kullanabilirsiniz, dizeyi çevreleyen alıntılarla eşleşmedikleri sürece:

    var answer1 = "Her şey yolunda"; // Çift tırnak içinde tek tırnak

    var answer2 = "Ona 'Johnny' deniyor"; // Çift tırnak içinde tek tırnak

    var answer3 = 'Ona "Johnny" denir'; // Tek tırnak içinde çift tırnak

    JavaScript Numaraları

    JavaScript'in yalnızca bir tür numarası vardır.

    Sayılar ondalıklı veya ondalıksız yazılabilir:

    var x1 = 34.00; // Ondalıklarla yazılmıştır

    var x2 = 34; // Ondalıksız yazılmıştır

    Ekstra büyük veya ekstra küçük sayılar bilimsel (üstel) gösterimle yazılabilir:

    var y = 123e5; // 12300000

    var z = 123e-5;  // 0.00123
