JavaScript Fonksiyon Sözdizimi

Bir JavaScript işlevi, Fonksiyon anahtar sözcüğüyle, ardından bir adla ve ardından parantez () ile tanımlanır.

Fonksiyon adları harf, rakam, alt çizgi ve dolar işareti içerebilir (değişkenlerle aynı kurallar).

Parantezler virgülle ayrılmış parametre adları içerebilir: (parametre1, parametre2, ...)

Fonksiyon tarafından yürütülecek kod, küme parantezlerinin içine yerleştirilir: {}

fonksiyon adı (parametre1, parametre2, parametre3) {

// yürütülecek kod

}

İşlev parametreleri, işlev tanımındaki parantez () içinde listelenir.

İşlev bağımsız değişkenleri, işlev çağrıldığında işlev tarafından alınan değerlerdir.

İşlevin içinde, bağımsız değişkenler (parametreler) yerel değişkenler gibi davranır.

Bir İşlev, diğer programlama dillerinde bir Prosedür veya Alt Program ile hemen hemen aynıdır.

İşlev Çağırma

"Bir şey" işlevi çağırdığında (çağırdığında) işlevin içindeki kod yürütülür:

  • Bir olay meydana geldiğinde (kullanıcı bir düğmeyi tıkladığında)

  • JavaScript kodundan çağrıldığında (çağrıldığında)

  • Otomatik olarak (kendi kendine çağrılır)

    İşlev Dönüşü

    JavaScript bir iade ifadesine ulaştığında, işlev yürütmeyi durdurur.

    İşlev bir ifadeden çağrıldıysa, JavaScript çağırma ifadesinden sonra kodu çalıştırmak için "geri döner".

    İşlevler genellikle bir dönüş değeri hesaplar. Dönüş değeri "arayan" a geri döndürülür:

    İki sayının çarpımını hesaplayın ve sonucu döndürün:

    var x = myFunction (4, 3); // İşlev çağrılır, dönüş değeri x ile biter


    function myFunction (a, b) {

    return * b; // Fonksiyon a ve b'nin çarpımını döndürür

    }

    X içindeki sonuç:

    12
