    Bir JavaScript değişkeninin veri türünü bulmak için typeof operatörünü kullanabilirsiniz.

    typeof "John" // "string" i döndürür

    typeof 3.14 // "sayı" döndürür

    typeof false // "boolean" değerini döndürür

    typeof [1,2,3,4] // "Nesne" yi döndürür

    typeof {name: 'John', yaş: 34} // "Nesne" yi döndürür

    typeof new Date () // "nesne" döndürür

    typeof function () {} // "Fonksiyon" döndürür

    typeof myCar // "tanımsız" döndürür *

    typeof null // "object" değerini döndürür

    Lütfen dikkat:

  • NaN veri türü sayıdır

  • Bir dizinin veri türü nesnedir

  • Bir tarihin veri türü nesnedir

  • Null veri türü nesnedir

  • Undefined değişkeninin veri türü undefined *

  • Değer atanmamış bir değişkenin veri türü de tanımsızdır *

    Typeof veri türü

    Typeof operatorü bir değişken değildir. Operatörlerin (+ - * /) herhangi bir veri türü yoktur.

    Ancak, typeof operatörü her zaman bir dize döndürür (işlenenin türünü içerir).

    JavaScript Türü Dönüştürme

    JavaScript değişkenleri yeni bir değişkene ve başka bir veri türüne dönüştürülebilir:

  • JavaScript işlevi kullanarak

  • JavaScript'in kendisi tarafından otomatik olarak

    • Sayıları Dizelere Dönüştürme

    Global yöntem String () sayıları dizelere dönüştürebilir.

    Herhangi bir sayı tipinde kullanılabilir, değişmez değerler, değişkenler veya ifadeler:

    String (x) // x sayı değişkeninden bir dize döndürür

    String (123) // tam sayı 123'ten bir dize döndürür

    String (100 + 23) // bir ifadeden bir sayıdan bir dize döndürür

    ToString () yönteminin Number yöntemi de aynısını yapar.

    x.toString ()

    (123) .ToString ()

    (100 + 23). ToString ()

    Yöntem Açıklama
    toExponential () Üstel gösterim kullanılarak yuvarlanmış ve yazılmış bir sayı olan bir dize döndürür.
    toFixed () Bir dize döndürür, bir sayı yuvarlanmış ve belirtilen ondalık sayılarla yazılmıştır.
    toPrecision () Belirtilen uzunlukta bir sayı yazılan bir dize döndürür

    Booleanları Dizelere Dönüştürme

    Global yöntem String (), boolean'ları dizelere dönüştürebilir.

    String (false) // "false" değerini döndürür

    String (true) // "true" değerini döndürür

    ToString () yöntemi Boole yöntemi de bunu yapar.

    false.toString () // "false" değerini döndürür

    true.toString () // "true" değerini döndürür

    Tarihleri ​​Dizelere Dönüştürme

    Genel yöntem String (), tarihleri ​​dizelere dönüştürebilir.

    String (Date ()) // "17 Temmuz 2014 Per 15:38:19 GMT + 0200 (W.Avrupa Yaz Saati)" döndürür

    Date yöntemi toString () de aynı şeyi yapar.

    Date (). ToString () // "17 Temmuz 2014 Per 15:38:19 GMT + 0200 (W.Avrupa Yaz Saati)" döndürür

    Dizeleri Sayılara Dönüştürme

    Global yöntem Number (), dizeleri sayılara dönüştürebilir.

    Sayı içeren dizeler ("3.14" gibi) sayılara dönüştürülür (3.14 gibi).

    Boş dizeler 0'a dönüşür.

    Başka bir şey NaN (Sayı Değil) biçimine dönüşür.

    Sayı ("3.14") // 3,14 değerini döndürür

    Sayı ("") // 0 döndürür

    Sayı ("") // 0 döndürür

    Sayı ("99 88") // NaN değerini döndürür

    Yöntem Açıklama
    parseFloat () Bir dizeyi ayrıştırır ve bir kayan nokta sayısı döndürür
    parseInt () Bir dizeyi ayrıştırır ve bir tam sayı döndürür

    Unary + Operatörü

    Unary + operatörü bir değişkeni sayıya dönüştürmek için kullanılabilir:

    var y = "5"; // y bir dizedir

    var x = + y; // x bir sayıdır


    Değişken dönüştürülemezse, yine de bir sayı olur, ancak NaN (Sayı Değil) değeriyle:

    var y = "John"; // y bir dizedir

    var x = + y; // x bir sayıdır (NaN)

    Otomatik Tip Dönüştürme

    JavaScript "yanlış" bir veri türünde çalışmaya çalıştığında, değeri "doğru" bir türe dönüştürmeye çalışır.

    Sonuç her zaman beklediğiniz gibi değildir:

    5 + null // null değerini 0'a dönüştürdüğü için 5 değerini döndürür

    "5" + null //, null "null" değerine dönüştürüldüğünden "5null" değerini döndürür

    "5" + 2 // "52" değerini döndürür, çünkü 2 "2" biçimine dönüştürülür

    "5" - 2 //, 3 değerini döndürür, çünkü "5", 5'e dönüştürülür

    "5" * "2" // 10 döndürür, çünkü "5" ve "2" 5 ve 2'ye dönüştürülür
